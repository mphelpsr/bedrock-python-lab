# Amazon Bedrock Python Lab 🚀

Este repositório foi criado para documentar meus estudos sobre IA Generativa utilizando o **Amazon Bedrock** com Python. O objetivo é explorar as APIs de diferentes modelos (como Claude, Llama ou Titan) e entender como integrar LLMs em aplicações reais.

## 🛠️ Tecnologias
* **Linguagem:** Python 3.x
* **SDK AWS:** Boto3
* **Modelos:** Amazon Bedrock (Anthropic, Meta, etc.)

## 📋 Pré-requisitos
Antes de rodar o projeto, você precisará de:
1. Uma conta na **AWS**.
2. Acesso aos modelos habilitado no console do **Amazon Bedrock** (Model Access).
3. Credenciais da AWS configuradas localmente (`aws configure`).
